Currency Converter Web App

A simple and responsive Currency Converter built using HTML, CSS, and JavaScript.
This application allows users to convert one currency to another using real-time exchange rates from an API.

Features
	•	Convert between multiple currencies
	•	Real-time exchange rates using API
	•	Country flag display for selected currencies
	•	Responsive design for desktop and mobile
	•	Clean and simple user interface

Technologies Used
	•	HTML5
	•	CSS3
	•	JavaScript
	•	Exchange Rate API

How to Use
	1.	Enter the amount you want to convert
	2.	Select the source currency
	3.	Select the target currency
	4.	Click the Convert button to see the result
